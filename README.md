This is an example application that shows good practices for development environments.

# Initial Setup

- Install Docker
- Install Git
- Install Node (nvm recommended)
- Clone Repo
- `npm i`
- `npm everything`

# Things to see

- View the [main UI](http://localhost:3000/)
- [List TODOs from the API](http://localhost:3001/todos)
